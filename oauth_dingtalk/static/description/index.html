Odoo中的钉钉设定

设置 - 技术 - 系统参数设置

web.base.url与钉钉回调的域名需相同

需新增一条web.base.url.freeze的设置冻结URL，避免URL自动变更

特别注意

  • dingtalk文件夹需要复制到Odoo根目录下，请至https://github.com/luss613/oauth_dingtalk获取dingtalk文件夹。
  • 因扫码登入需注册web服务接收钉钉回调，因此在多账套（数据库）状况下无法使用此模块。
  • 钉钉绑定手机如非大陆手机，手机号码输入时不带国码，如手机号码首位为0，则略过输入0。

适用于Odoo 13.0
